Dziedziczenie z klasy bazowej

Bardzo ważnym mechanizmem programowania obiektowego jest tzw. dziedziczenie, czyli utworzenie nowej klasy na bazie istniejącej i rozbudowanie jej o nowe dane i funkcje. Wykorzystajmy klasę Osoba do stworzenia klasy Uczen, która dodatkowo będzie zawierać zmienną klasa.

class Osoba:
imie = "imie"
nazwisko = "nazwisko"
wiek = 0
def __init__(self, imie, nazwisko, wiek):
self.imie = imie
self.nazwisko = nazwisko
self.wiek = wiek
def wypisz(self):
print(self.imie,self.nazwisko,self.wiek)

class Uczen(Osoba):
klasa = "0"
uczen1 = Uczen("Anna","Nowak",16)
print(vars(uczen1))
{'imie': 'Anna', 'nazwisko': 'Nowak', 'wiek': 16, 'klasa': '0'}

Info: Klasę, która została utworzona w oparciu o inną klasę określa się mianem klasa pochodna (lub podklasą od ang. subclass), a klasę, po której nastąpiło dziedziczenie, klasa bazowa (lub nadklasą ang. superclass).

Aby zmienić wartość zmiennej klasa w nowym obiekcie możemy wykorzystać kropkę lub funkcję setattr():

uczen1.klasa = "3a"
print(vars(uczen1))
{'imie': 'Anna', 'nazwisko': 'Nowak', 'wiek': 16, 'klasa': '3a'}

Najwygodniej jednak byłoby zmienić konstruktor, tak, żeby uwzględniał nowy atrybut. Możesz to zrobić pisząc nową funkcję __init__() w klasie Uczen:

class Uczen(Osoba):
klasa = "0"
def __init__(self, imie, nazwisko, wiek,klasa):
self.imie = imie
self.nazwisko = nazwisko
self.wiek = wiek
self.klasa = klasa
uczen1 = Uczen("Anna","Nowak",16,"3a")
print(vars(uczen1))
{'imie': 'Anna', 'nazwisko': 'Nowak', 'wiek': 16, 'klasa': '3a'}

Jeszcze łatwiej jest wykorzystać konstruktor klasy bazowej. Do klasy bazowej odwołujemy się funkcją super() i przekazujemy do __init__() parametry, które ustawia konstruktor klasy bazowej.

klasa = "0"
def __init__(self, imie, nazwisko, wiek, klasa):
super().__init__(self.imie,self.nazwisko,self.wiek)
self.klasa = klasa
uczen1 = Uczen("Anna","Nowak",16,"3a")
print(vars(uczen1))
{'imie': 'Anna', 'nazwisko': 'Nowak', 'wiek': 16, 'klasa': '3a'}

Sprawdź czy potrafisz

Wykorzystaj klasę Komorka do utworzenia klasy WlascielKomorki. Oprócz pól, które znajdują się w klasie komórka nowa klasa ma pola wlasciciel = ”” oraz numer = 0. Wykorzystując konstruktor klasy bazowej napisz konstruktor dla nowej klasy, tak, by można było ustawić wszystkie argumenty. Za pomocą konstruktora utwórz przykładowy obiekt: Samsung S7 2016 Ala 606505404. Na końcu wypisz zawartość obiektu.

Jak to zrobić?

Na bazie najprostszej klasy Komorka z początku tego modułu utwórz nową klasę WlascicielKomorki(Komorka). Dopisz pola self.wlasciciel = ”” oraz self.numer = 0. Dopisz funkcję __init()__, która odwołuje się do funkcji klasy bazowej super().__init__() i przesyła do niej self.producent, model oraz rok_produckji. Na końcu utwórz obiekt WlascicileKomorki("Samsung","S7",2016,"Ala",606505404) i korzystając z funkcji vars() wypisz jego zawartość.

class Komorka:
producent=""
model=""
rok_produkcji=0
def __init__( self, producent, model, rok_produkcji):
self.producent = producent
self.model = model
self.rok_produkcji = rok_produkcji
class WlascicielKomorki(Komorka):
wlasciciel = ""
numer = 0
def __init__( self, producent, model, rok_produkcji, wlasciciel, numer):
super().__init__(self.producent, self.model, self.rok_produkcji)
self.wlasciciel = wlasciciel
self.numer = numer
wlasc1 = WlascicielKomorki("Samsung", "S7", 2016, "Ala",606505404)
print(vars(wlasc1))
