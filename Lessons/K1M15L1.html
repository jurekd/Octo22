Definiowanie funkcji

Zdefiniujmy prostą funkcję, która wypisuje coś na ekranie ‘Witaj!’ i nazwijmy ją witaj(). Funkcję definiujemy słowem kluczowym def:

def witaj():
print("Witaj!")

witaj() Witaj!

Zdefiniowaną wcześniej funkcję wywołujesz wpisując jej nazwę. Jeśli funkcja nie posiada parametrów, to po jej nazwie umieszczamy puste nawiasy.

Określmy teraz funkcję, która przyjmuje parametr. Niech to będzie funkcja wypisz(lista), która wypisuje podaną listę, rozdzieloną przecinkami:

def wypisz(lista):
for element in lista:
print(element,end=',')

liczby = [3,5,1,2]
wypisz(liczby) Witaj!

Jeśli wywołując funkcję nie podamy oczekiwanego argumentu, wówczas wystąpi błąd:

wypisz() TypeError: wypisz() takes exactly 1 arguments (0 given)

W komunikacie o błędzie zostaniemy poinformowani ile argumentów przyjmuje funkcja.

Możemy uniknąć takiej sytuacji, że ktoś nie poda argumentu, określając wartości domyślne dla parametrów, które zostaną pominięte. W naszym przypadku jeśli ktoś nie poda listy, możemy przyjąć pustą listę.

def wypisz(lista=[]):
for element in lista:
print(element,end=',')

wypisz()

Teraz nie mamy błędu – pętla for nie zostanie wykonana, bo lista jest pusta, zatem na wyjściu nie pojawi się nic. Jeśli chcielibyśmy poinformować użytkownika o tym fakcie, należałoby sprawdzić czy argument jest pustą listą i wypisać stosowny komunikat:

def wypisz(lista=[]):
if lista!=[]:
for element in lista:
print(element,end=',')
else:
print('Lista jest pusta!')

wypisz()
Lista jest pusta!

Sprawdź czy potrafisz

Napisz funkcję witaj, która przyjmuje imię osoby jako parametr i wyświetla komunikat np. ‘Witaj Tomek !’. Jeżeli imię nie zostanie podane, wypisywany jest komunikat ‘Nie podano imienia’. Sprawdź działanie funkcji podając imię Tomek.

Jak to zrobić?

Zdefiniuj funkcję z argumentem domyślnym witaj(imie=''). Instrukcją if sprawdź czy imie nie jest puste. Jeśli nie jest, wypisz komunikat łącząc napis ‘Witaj’, zmienną imie i znak ‘!’. Jeśli jest puste, po else wypisz komunikat ‘Nie podano imienia’.

def witaj(imie=''):
if imie!='':
print('Witaj',imie,'!')
else:
print('Nie podano imienia')

witaj('Tomek')
