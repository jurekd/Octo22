Definiowanie klas

Załóżmy, że w naszej aplikacje chcemy przechowywać dane o osobach. Musimy więc zdefiniować wzorzec takich danych. W nowoczesnych językach programowania taki wzorzec określa się mianem klasy. Zdefiniujmy zatem pierwszą prostą klasę w języku Python:

class Osoba:
imie = "imie"
nazwisko = "nazwisko"
wiek = 0
print(Osoba)

Klasa jest wzorcem obiektu, zatem nie można wypisać bezpośrednio jej zawartości – wypisywana jest nazwa klasy (typ). Nasza klasa ma trzy zmienne (atrybuty): imie, nazwisko oraz wiek.

Natomiast możemy wypisać jej składowe:

print(Osoba.imie,Osoba.nazwisko,Osoba.wiek)
imie nazwisko 0

Utwórzmy teraz dwa konkretne egzemplarze tej klasy (realne byty), zwane obiektami:

osoba1= Osoba()
osoba1.imie = "Kamil"
osoba1.nazwisko = "Bilski"
osoba1.klasa = 15
osoba2= Osoba()
osoba2.imie = "Ala"
osoba2.nazwisko = "Nowak"
osoba2.klasa = 16
print(osoba1.imie,osoba1.nazwisko,osoba1.klasa)
print(osoba2.imie,osoba2.nazwisko,osoba2.klasa)
Kamil Bilski 15
Ala Nowak 16

Takie tworzenie obiektów jest stosunkowo niewygodne, bowiem każdą daną w klasie, trzeba wprowadzać osobno. Zamiast tego możemy wykorzystać funkcję, która ustawi nam wszystkie konieczne dane w momencie tworzenia obiektu. Ta funkcja zwana jest konstruktorem i w języku Python ma nazwę __init__() (przed i po słowie init są 2 podkreśliniki):

class Osoba:
imie = "imie"
nazwisko = "nazwisko"
wiek = 0
def __init__(self, imie, nazwisko, wiek):
self.imie = imie
self.nazwisko = nazwisko
self.wiek = wiek

Mając zdefiniowany konstruktor możemy dużo łatwiej utworzyć obiekty.

osoba1 = Osoba("Kamil","Bilski",15)
osoba2 = Osoba("Anna", "Nowak", 16)
print(osoba1.imie,osoba1.nazwisko,osoba1.klasa)
print(osoba2.imie,osoba2.nazwisko,osoba2.klasa)
Kamil Bilski 15
Ala Nowak 16

Sprawdź czy potrafisz

Zdefiniuj klasę Komorka, w której będą przechowywane następujące dane: producent, model, i rok_produkcji. W klasie zdefiniuj konstruktor w celu łatwiejszego tworzenia obiektów. Na koniec utwórz przynajmniej jeden obiekt z danymi: Samsung, S7, 2016 i wypisz jego zawartość.

Jak to zrobić?

W class Komorka utwórz zmienne producent="", model="", rok_produkcji=0. Zdefiniuj też funkcję __init__(self, producent, model, rok_produkcji). Następnie utwórz obiekt z podanymi danymi i wypisz wszystkie dane z obiektu.

class Komorka:
producent=""
model=""
rok_produkcji=0
def __init__( self, producent, model, rok_produkcji):
self.producent = producent
self.model = model
self.rok_produkcji = rok_produkcji
tel1 = Komorka("Samsung", "S7", 2016)
print(tel1.producent, tel1.model, tel1.rok_produkcji)
