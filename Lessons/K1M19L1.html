Definiowanie wyrażeń regularnych

Wyrażenia regularne służą do definiowania wzorców i sprawdzania czy jakiś tekst pasuje do tego wzorca. Dzięki temu możemy np. sprawdzić czy podany tekst jest kodem pocztowym. W języku Python mamy do tego celu dwie funkcje z modułu re: match() – sprawdza tekst od jego początku i search() – sprawdza tekst od dowolnego znaku w środku.

Wykorzystajmy najpierw funkcję match() do sprawdzenia kodu pocztowego:

import re
kod1 = '00-950'
kod2 = '123-254'
wzorzec = '[0-9]{2}-[0-9]{3}'
if re.match(wzorzec,kod1):
print('kod1 jest ok')
else:
print('kod1 jest błędny')
if re.match(wzorzec,kod2):
print('kod2 jest ok')
else:
print('kod2 jest błędny')
kod1 jest ok
kod2 jest błędny

Powyższy wzorzec ‘[0-9]{2}-[0-9]{3}’ oznacza, że dopuszczalne są 2 dowolne cyfry, kreska i 3 dowolne cyfry. Alternatywnie moglibyśmy go zapisać tak: ‘\d{2}-\d{3}’, gdzie \d oznacza dowolną cyfrę.

Wzorce mogą się odnosić także do liter w tekstach, przyjmijmy, że np. nazwisko musi zaczynać się dużą literą i mieć co najmniej 3 znaki:

import re
nazwisko1 = 'ala'
nazwisko2 = 'Nowak'
wzorzec = '[A-Z][a-z]{2}'
if re.match(wzorzec,nazwisko1):
print('nazwisko1 jest ok')
else:
print('nazwisko1 jest błędne')
if re.match(wzorzec,nazwisko2):
print('nazwisko2 jest ok')
else:
print('nazwisko2 jest błędne')
nazwisko1 jest błędne
nazwisko2 jest ok

Wzorzec [A-Z] oznacza dowolną literę od A do Z, [a-z] litery od a do z, a w nawiasach klamrowych jest liczba znaków, która musi być zgodna.

Można tworzyć też bardziej skomplikowane wzorce, np. sprawdzający czy podany adres jest adresem e-mail:

import re
email1= 'info@biuro.pl'
email2= 'kontakt@adres*wp.pl'
wzorzec = '^[a-z]+@([a-z].)+[a-z]{2,3}$'
if re.match(wzorzec, email1):
print('email1 jest ok')
else:
print('email1 jest błędny')
if re.match(wzorzec, email2):
print('email2 jest ok')
else:
print('email2 jest błędny')
email1 jest ok
email2 jest błędny

Powyższy wzorzec oznacza: [a-z]+ co najmniej jedna litera od a do z, małpa (@) co najmniej jedna kombinacja liter z kropą [a-z]. oraz na końcu 2 lub 3 litery od a do z. Znak ^ na początku oraz $ na końcu, oznaczają że tekst musi się zaczynać i kończyć zgodnie z wzorcem, a nie może być tylko jego częścią.

Info: Nie przejmuj się tym, że niektóre wyrażenia wyglądają tak skomplikowanie. Wiele przykładów gotowych wzorców znajdziesz w Internecie.

Sprawdź czy potrafisz

Napisz program, który z podanej listy telefonów [‘+48200111’,’+500600’,’600123’, ‘+48144288’] wypisze tylko telefony, które mają numer kierunkowy z przodu np. +48200111.

Jak to zrobić?

Zdefiniuj listę telefonów. W pętli for po wszystkich elementach listy sprawdź czy element jest zgodny z wzorcem '\+[0-9]{8}’ i jeśli jest wypisz numer telefonu.

import re
telefony = ['+48200111','+500600','600123', '+48144288']
wzorzec = '\+[0-9]{8}'
for telefon in telefony:
if re.match(wzorzec,telefon):
print(telefon)
