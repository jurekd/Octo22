Zwrot wartości przez funkcjęt

Funkcja może zwrócić wartość po słowie return. Napiszmy prostą funkcję suma, która pobiera dwie liczby i zwraca ich sumę:

def suma(a, b):
c = a + b
return c

wynik = suma(2,3)
print(wynik)
5

W funkcji można określić wiele instrukcji return. Każda z nich kończy działanie funkcji i zwraca podaną po return wartość:

def bezwzgledna(liczba):
if liczba>=0:
return liczba
else:
return -liczba

print(bezwzgledna(-3))
3

Funkcja może też kończyć swoje działanie słowem return, nie zwracając żadnej wartości:

def bezwzgledna(liczba):
def dodatnia(liczba):
if liczba>0:
print("Liczba jest dodatnia")
return
print("Liczba jest ujemna")
return

dodatnia(10)
Liczba jest dodatnia

Jeśli będziemy chcieli przypisać do zmiennej wynik działania funkcji, a funkcja zwróci „puste” return, zmienna przyjmie wartość None.

def odwrotnosc(liczba):
if liczba == 0:
return
return 1/liczba

wynik = odwrotnosc(0)
print(wynik)
None

Info: Jeżeli funkcja nie zawiera ani jednej instrukcji return, jak w poprzedniej lekcji, również zwróci wartość None.

Sprawdź czy potrafisz

Napisz funkcję znaki(tekst), która sprawdza ile jest znaków w podanym tekście, pomijając spacje. Jeżeli użytkownik nie poda tekstu do sprawdzenia, zwracane jest 0. Przetestuj działanie funkcji dla argumentu „Ala ma kota”.

Jak to zrobić?

Zdefiniuj funkcję z argumentem domyślnym o nazwie znaki(tekst=''). Sprawdź czy tekst jest pusty. Jeśli jest, zwróć 0. Jeśli nie, to ustaw licznik znaków na 0, a następnie w pętli for zwiększaj licznik znaków, jeżeli różnią się od spacji. Zwróć wyliczoną wartość.

def znaki(tekst=''):
if tekst == '':
return 0
licznik = 0
for znak in tekst:
if znak!=' ':
licznik = licznik +1
return licznik

ile = znaki("Ala ma kota")
print(ile)
