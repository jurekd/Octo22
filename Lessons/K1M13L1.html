Moduł time

Zmienne zawierające Datę i czas w języku Python można używać na kilka sposobów. Najpopularniejszym z nich jest zastosowanie modułu time.

Czas w module time liczony jest jako liczba sekund, która upłynęła od 1 stycznia 1970 roku Python zapisuje datę i czas w specjalnej strukturze.

Różnicę w sekundach od 1.01.1970 r. można uzyskać przez wywołanie funkcji time():

import time
sekundy = time.time()
print(sekundy) 1501404121.15

Aby wyświetlić bieżącą datę i czas lokalny (tj. zgodny ze strefą czasową, w której się znajdujesz), możesz użyć funkcji ctime():

import time
data = time.ctime()
print(data) Fri May 12 10:34:55 2017

Jeśli chcesz wyświetlić czas w innym formacie, użyj funkcji strftime(), podając jako argument formatowanie (%Y – oznacza rok, %m – oznacza dzień, %d – oznacza dzień, %H – oznacza godzinę, %M – oznacza minuty) oraz strukturę czasu (możesz ją pominąć, wówczas zostanie podstawiony bieżąca data i czas).

import time
data = time.strftime('%Y-%m-%d')
czas = time.strftime('%H:%M',(2017,10,10,12,15,1,1,1,1))
print(data,czas) 2017-05-12 10:35

Dzięki funkcji localtime() możesz zobaczyć datę i czas (sekundy, które upłynęły od 1.01.1970) w postaci specjalnej struktury, z której będziesz mógł wydobyć poszczególne składowe:

import time
data = time.localtime()
print(data) time.struct_time(tm_year=2017, tm_mon=5, tm_mday=12, tm_hour=10, tm_min=56, tm_sec=13, tm_wday=4, tm_yday=132, tm_isdst=1)

Mając strukturę czasu można z łatwością wydobyć z niej np. rok, miesiąc czy dzień.

import time
data = time.localtime()
print(data.tm_year,data.tm_mon,data.tm_mday) 2017 5 12

Funkcja strptime() przyjmująca datę zapisaną jako tekst oraz jej format, pozwala na ustawienie daty na podaną wartość, np. 1 stycznia 2017 r.

import time
mojadata = time.strptime('2017-12-01','%Y-%m-%d')
print(mojadata) time.struct_time(tm_year=2017, tm_mon=12, tm_mday=1, tm_hour=0, tm_min=0, tm_sec=0, tm_wday=4, tm_yday=335, tm_isdst=0)

Strukturę możesz zamienić na sekundy, które upłynęły od 1.01.1970 r. korzystając z funkcji maketime().

import time
mojadata = time.strptime('2017-12-01','%Y-%m-%d')
print(time.mktime(mojadata)) 1514674800.07

Sprawdź czy potrafisz

Wypisz bieżąco datę 2017-12-24 w postaci ‘Dzisiaj jest 24.12 roku 2017’.

Jak to zrobić?

Utwórz datę 2017-12-24 wykorzystując funkcję strptime(). Następnie użyj funkcji strftime, żeby sformatować datę w formacie 'Dzisiaj jest %m.%d roku %Y'.

import time
mojadata = time.strptime('2017-12-24','%Y-%m-%d')
data = time.strftime('Dzisiaj jest %m.%d roku %Y',mojadata)
print(data)
