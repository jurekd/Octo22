Więcej o parametrach funkcjit

Rozważmy funkcję iloraz dzielącą podane dwie liczby podane jako parametry: dzielna (liczba która będzie dzielona) i dzielnik (liczba przez którą nastąpi dzielenie):

def iloraz(dzielna, dzielnik):
rezultat = dzielna/dzielnik
return rezultat

wynik = iloraz(1,2)
print(wynik)
0.5

Jeśli argumenty przekażemy w odwrotnej kolejności, wynik będzie już inny:

wynik = iloraz(2,1)
print(wynik)
2

A co jeśli chcielibyśmy podawać argumenty funkcji w dowolnej kolejności? W języku Python jest to możliwe, tylko musisz nazwać argumenty:

wynik = iloraz(dzielnik=2,dzielna=1)
print(wynik)
0.5

Niektóre parametry funkcji mogą być opcjonalne. Rozważmy funkcję suma, która liczy sumę dwóch lub opcjonalnie 3 liczb:

def suma(a, b, c=None):
wynik = a+b
if c != None:
wynik = wynik +c
return wynik

print(suma(2,3))
5
print(suma(2,3,5))
10

Co więcej Python umożliwia określenie dowolnej liczby argumentów. Przekazywane są one w krotce, której nazwę umieszczamy po znaku ‘*’:

def suma(*liczby):
suma = 0
for liczba in liczby:
suma = suma + liczba
return suma

print(suma(1,2,3,4,5))
15

Sprawdź czy potrafisz

Rozwiń funkcję zliczającą znaki z poprzedniej lekcji o możliwość określenia czy spacje mają być zliczane czy nie. Ten drugi argument o nazwie spacje jest opcjonalny i jeśli nie zostanie podany, to zliczane są znaki bez spacji. Jeśli natomiast spacje przyjmą wartość ‘Tak’, wówczas są także zliczane. Przetestuj działanie funkcji ponownie na tekście „Ala ma kota”, określając, że spacje mają być zliczane. Postaraj się podać argumenty funkcji odnosząc się do ich nazw tj. znaki(spacje="Tak", tekst="Ala ma kota")

Jak to zrobić?

Zdefiniuj funkcję znaki z argumentami tekst='' oraz spacje=None. Sprawdź czy tekst jest pusty. Jeśli jest, zwróć 0. Jeśli nie, to sprawdź czy spacje==’Tak’. Jeśli tak, to zwróć długość tekstu – funkcja len(), jeśli nie, to ustaw licznik znaków na 0, a następnie w pętli for zwiększaj licznik znaków i na końcu zwróć wyliczoną wartość.

def znaki(tekst='',spacje=None):
if tekst == '':
return 0
if spacje == 'Tak':
return len(tekst)
licznik = 0
for znak in tekst:
if znak!=' ':
licznik = licznik +1
return licznik

ile = znaki(spacje="Tak", tekst="Ala ma kota")
print(ile)
