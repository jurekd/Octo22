Przerywanie i zagnieżdżanie pętli for

Pętla for wykonywana jest od pierwszego do ostatniego elementu jakiegoś zbioru. Możesz ją jednak przerwać instrukcją break.

for x in range(1,7)
if x==5:
break
print(x) 1
2
3
4

Możesz też pominąć wykonanie instrukcji w pętli i przejść do kolejnego elementu w zbiorze. Służy to tego instrukcja continue:

for x in range(1,7)
if x==5:
continue
print(x) 1
2
3
4
6

Instrukcja for, podobnie jak instrukcja if może zawierać blok else. Blok ten wykonywany jest po wyczerpaniu wszystkich elementów w pętli. Poniższy przykład prezentuje program sprawdzający czy liczba 111 jest liczbą pierwszą.

liczba = 111
for dzielnik in range(2,liczba//2):
if liczba%dzielnki == 0:
print('Liczba nie jest pierwsza')
break
else:
print('Liczba pierwsza')

Pętle można zagnieżdżać jedna w drugiej. Przykładowo chcemy narysować prostokąt o bokach 3 i 5.

for bok1 in range(1,4):
for bok2 in range(1,6):
print('*',end='')
print()

*****
*****

Info: Przekazanie argumentu end='' do instrukcji print powoduje, że zamiast przejścia do nowej linii, kolejne wartości będą wypisywane w tym samych wierszu. Przekazanie jakiegoś znaku po end, np. end = ',' spowoduje rozdzielenie wartości tym znakiem (tutaj: przecinkami).

Sprawdź czy potrafisz

Wypisz wszystkie liczby pierwsze w zakresie od 2 do 99. Liczby rozdziel przecinkami.

Jak to zrobić?

Wykorzystaj dwie pętle for – jedną dla liczba z zakresu od 2 do 100, drugą od 2 do danej liczby dzielonej całkowicie przez 2. Druga pętla zawiera kod analogiczny do przykładu zawartego w tej lekcji, z tym, że informacja o tym, że liczba nie jest pierwsza nie jest wypisywana, a w przypadku jej znalezienia, wypisywana jest właśnie ta liczba.

for liczba in range(2,30):
for dzielnik in range(2,liczba//2):
if liczba%dzielnik == 0:
break
else:
print(liczba,end=','))
