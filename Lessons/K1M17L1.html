Używanie wyrażeń listowych

Wyrażenia listowe pozwalają na skrócony zapis przeglądania elementów listy w pętli for i wykonywania na nich filtrowania (odpowiednik instrukcji if) oraz przekształcania elementów zgodnie z podaną funkcją.

Rozważmy jeszcze raz przykład tworzący listę kwadratów dla podanej listy liczb:

liczby = [1,2,3,4,5] # krócej: range(1, 6)
kwadraty = []
for x in liczby:
kwadraty.append(x*x)
print(kwadraty)
[1, 4, 9, 16, 25]

Wykorzystując wyrażenia listowe możesz powyższy kod może zapisać w następujący sposób:

liczby = [1,2,3,4,5]
kwadraty = [x*x for x in liczby]
print(kwadraty)
[1, 4, 9, 16, 25]

Zwróć uwagę, że działa to tak samo jak mapowanie z wykorzystaniem wyrażeń lambda:

kwadraty = map(lambda x: x*x, liczby)

Odfiltrujmy teraz liczby parzyste. Najpierw klasycznie, przy użyciu pętli for:

liczby = [1,2,3,4,5]
parzyste = []
for x in liczby:
if x%2 == 0:
parzyste.append(x)
print(parzyste)
[2, 4]

To samo możesz zapisać w jednej linijce jako wyrażenie listowe:

liczby = [1,2,3,4,5]
parzyste= [x for x in liczby if x%2==0]
print(parzyste)
[2, 4]

Działa to analogicznie jak funkcja filtrowania z wyrażaniem lambda:

parzyste = filter(lambda x: x%2 == 0, liczby)

Ja powiedziano na wstępie stosując wyrażenia listowe możesz w łatwy sposób połączyć filtrowanie i mapowanie. Policzmy kwadraty, ale tylko liczb parzystych:

liczby = [1,2,3,4,5]
kwadraty_parzystych = [x*x for x in liczby if x%2==0]
print(parzyste)
[4, 16]

Gdybyśmy chcieli użyć lambda wyrażeń, musielibyśmy połączyć funkcje map i filter:

kwadraty_parzystych = map(lambda x: x*x, filter(lambda x: x%2 == 0, liczby))

Powyższy zapis jest mniej czytelny niż wyrażenie listowe, poza tym wykorzystasz go tylko w najnowszej, trzeciej wersji języka Python.

Sprawdź czy potrafisz

Wykorzystując wyrażenia listowe przekształć listę telefony=['200123', '404111', '222333a', 'b500707', '120120'] do listy liczb, ale tylko dla tych numerów, dla których zapisany tekst da się przekształcić na liczbę.

Jak to zrobić?

Zdefiniuj listę telefonów. Jako wyrażenie mapujące użyj przekształcenia testu do liczby całkowitej int(x), a w filtrze, po if, wykorzystaj funkcję x.isdigit(). Wypisz utworzoną listę.

lista = ['200123', '404111', '222333a', 'b500707', '120120']
telefony = [int(x) for x in lista if x.isdigit()]
print(telefony)
