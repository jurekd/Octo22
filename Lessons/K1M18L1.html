Obsługa błędówt

Standardowo wiele sytuacji może prowadzić do wystąpienia błędu w języku Python: próba dzielenia przez zero, podanie indeksu wykraczającego poza rozmiar listy, zbioru czy krotki, brak możliwości konwersji tekstu np. na liczbę całkowitą i wiele innych.

a = 10/0
ZeroDivisionError: integer division or modulo by zero
lista = [1,2]
b = lista[3]
IndexError: list index out of range
tekst = '111a'
liczba = int(tekst)
ValueError: invalid literal for int() with base 10: '111a'

W swoim programie możesz przewidzieć wystąpienie błędów i odpowiednio na nie zareagować, np. wypisując odpowiedni komunikat. W tym celu fragment kodu, który może spowodować błąd umieszczasz w po słownie try, a błąd przechwytujesz po słownie except:

try:
a=10/0
except:
print("Błąd!")
Błąd!

Kod który będzie wykonywany, jeżeli błąd nie wystąpi możemy umieścić po słowie else:

try:
lista = [1,2]
x = lista[1]
except:
print("Błąd!")
else:
print(x)
2

Jeżeli po słowie except nie określimy nazwy wyjątku, który jest przechwytywany, łapane są wszystkie wyjątki. Możesz tylko łapać określone wyjątki i reagować wiedząc jaka jest przyczyna błędu:

try:
tekst = '111a'
liczba = int(tekst)
except ValueError:
print('To nie jest liczba!')
To nie jest liczba!

W powyższym kodzie nie zostanie złapany inny wyjątek niż związany ze złą wartością poddaną konwersji. Jeżeli chcielibyśmy obsłużyć inny wyjątek, trzeba dodać kolejny blok except:

try:
tekst = '222'
liczba = int(tekst)
wynik = liczba/0
except ValueError:
print('To nie jest liczba!')
except ZeroDivisionError:
print('Nie można dzielić przez 0!')
Nie można dzielić przez 0!

Możesz też łączyć obsługę kilku błędów w jednym wyrażeniu except, umieszczając je po przecinku w nawiasie okrągłym:

try:
tekst = '222'
liczba = int(tekst)
wynik = liczba/0
except(ValueError, ZeroDivisionError):
print('Wystąpił problem z obliczeniami!')
'Wystąpił problem z obliczeniami!'

Sprawdź czy potrafisz

Napisz program, który pobiera od użytkownika dwie liczby: dzielną i dzielnik, zamienia je na liczby całkowite i wypisuje wynik dzielenia. Jeżeli użytkownik poda złe liczby, wypisywany jest komunikat: "Podałeś złe liczby!", jeżeli druga będzie zerem, to wypisywany jest komunikat: "Druga liczba nie może być 0!". Zasymuluj działanie programu podając złą liczbę.

Jak to zrobić?

Za pomocą funkcji input() pobierz dzielną i dzielnik pod dwie zmienne, np. tekst_a i tekst_b. Następnie po try spróbuj zamienić je na liczby całkowite funkcją int(). Oblicz wynik dzieląc jedną liczbę przez drugą. W except złap wyjątek ValueError i wypisz komunikat "Podałeś złe liczby!", a w następnym except złap ZeroDivisionError i wypisz komunikat "Druga liczba nie może być 0!". Po else wypisz wynik.

tekst_a = input('Podaj dzielną')
tekst_b = input('Podaj dzielnik')
try:
liczba_a = int(tekst_a)
liczba_b = int(tekst_b
wynik = liczba_a/liczba_b
except ValueError:
print("Podałeś złe liczby!")
except ZeroDivisionError:
print("Druga liczba nie może być 0!")
else:
print(wynik)
