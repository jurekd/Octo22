Atrybuty obiektów

Obiekty tworzone są na podstawie wzorca klasy, ale w języku Python można później taki obiekt modyfikować, zmieniając jego składowe. W przykładzie utworzymy obiekt, a później dodamy do niego atrybut miasto.

class Osoba:
imie = "imie"
nazwisko = "nazwisko"
wiek = 0
def __init__(self, imie, nazwisko, wiek):
self.imie = imie
self.nazwisko = nazwisko
self.wiek = wiek
def wypisz(self):
print(self.imie,self.nazwisko,self.wiek)
osoba1 = Osoba("Kamil","Bilski",15)
osoba1.miasto = "Warszawa"
print(osoba1.nazwisko,osoba1.miasto)
Bilski Warszawa

Info: Podobny model obiektowości został zaimplementowany w języku JavaScript i nazywany jest prototypowaniem – na początku tworzony jest bowiem tylko prototyp obiektu.

Dopisanie składowej do obiektu nie oznacza, że odtąd wszystkie inne obiekty klasy będą go posiadały.

osoba2 = Osoba("Anna","Nowak",16)
print(osoba2.miasto)
AttributeError: 'Osoba' object has no attribute 'miasto'

Jeżeli chcemy dopisać jakąś funkcję lub atrybut do wszystkich obiektów, musimy dodać je do klasy, nie obiektu:

Osoba.miasto="brak"
osoba2 = Osoba("Anna","Nowak",16)
print(osoba2.miasto)
brak

Python dostarcza kilka użytecznych funkcji do operowania na atrybutach. Pierwszą z nich jest hasattr(), która sprawdza czy w podanej klasie lub obiekcie występuje dany atrybut:

osoba3 = Osoba("Iza","Maj",15)
print(hasattr(osoba3,'imie'))
print(hasattr(osoba3,'kod'))
True
False

Możemy też usunąć określony atrybut z obiektu – służy do tego funkcja delattr().

osoba3 = Osoba("Iza","Maj",15)
delattr(osoba3,'imie')
print(osoba3.imie)
AttributeError: Osoba instance has no attribute 'imie'

Python dostarcza też funkcje do pobierania wartości atrybutów oraz ich ustawiania. Służą do tego odpowiednio funkcje getattr() i setattr():

osoba3 = Osoba("Iza","Maj",15)
setattr(osoba3,'wiek',16)
print(getattr(osoba3,'wiek'))
print(osoba3.wiek)
16
16

Info: Funkcje getattr() oraz setattr() możesz stosować wymiennie z kropą.

Sprawdź czy potrafisz

Na bazie klasy Komorka utwórz obiekt Samsung S7 0 0. Następnie usuń z niego atrybut rok_produkcji, a dodaj za to atrybut kolor z wartością ‘biały’. Wypisz wszystkie atrybuty wykorzystując funkcję vars().

Jak to zrobić?

Na bazie najprostszej klasy Komorka z początku tego modułu utwórz obiekt np. o nazwie tel1 z wartościami Samsung S7 0 . Funkcją delattr() usuń z tego obiektu rok_produkcji. Dopisz tel1.kolor = "biały". Na końcu wypisz wynik funkcji vars(), której argumentem jest obiekt tel1.

class Komorka:
producent=""
model=""
rok_produkcji=0
def __init__( self, producent, model, rok_produkcji):
self.producent = producent
self.model = model
self.rok_produkcji = rok_produkcji
tel1 = Komorka("Samsung", "S7", 0)
delattr(tel1,'rok_produkcji')
tel1.kolor = "biały"
print(vars(tel1))
