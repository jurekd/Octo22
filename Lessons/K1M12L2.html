Operacje na zbiorach

Podstawową operacją jest suma, czyli złączenie zbiorów:

Sumę zbiorów możesz policzyć funkcją union() lub operatorem | (w naszym systemie ten operator nie zadziała).

A = {'lody', 'sok', 'banany', 'pomarańcze'}
B = {'banany', 'jabłka'}
C = A.union(B)
print(C) {'lody', 'sok', 'banany', 'pomarańcze', 'jabłka'}

Kolejną operacją jest przecięcie zbiorów, czyli znalezienie ich części wspólnej:

Przecięcie zbiorów możesz policzyć funkcją intersection() lub operatorem & (w naszym systemie ten operator nie zadziała).

A = {'lody', 'sok', 'banany', 'pomarańcze'}
B = {'banany', 'jabłka'}
C = A.intersection(B)
print(C) {'banany'}

Następna operacja to różnica zbiorów, czyli odjęcie elementów jednego zbioru od drugiego:

Różnicę zbiorów możesz policzyć funkcją difference() lub operatorem - (w naszym systemie ten operator nie zadziała).

A = {'lody', 'sok', 'banany', 'pomarańcze'}
B = {'banany', 'jabłka'}
C = A.difference(B)
print(C) {'lody', 'sok', 'pomarańcze'}

W języku Python możesz także policzyć różnicę symetryczną, czyli sumę zbiorów bez ich części wspólnej. Służy do tego funkcja symmetric_difference() lub operator ^:

A = {'lody', 'sok', 'banany', 'pomarańcze'}
B = {'banany', 'jabłka'}
C = A.symmetric_difference(B)
print(C) { 'lody', 'sok', 'pomarańcze', 'jabłka'}

Sprawdź czy potrafisz

Dwie osoby przygotowało listę gości na przyjęcie, które wspólnie organizują. Zbiór o nazwie spis1 zawiera imiona {‘Ala’,’Tomek’,’Iza’,’Piotrek’}, a zbiór spis2 zawiera imiona {‘Tomek’,’Beata’,’Kamil’}. Iza i Piotrek powiadomili, że nie mogą uczestniczyć w przyjęciu. Utwórz zbiór gości, w którym żadne imiona się nie powtarzają i nie zawiera Izy oraz Piotrka.

Jak to zrobić?

Utwórz zbiór spis1 oraz spis2 z podanymi imionami. Utwórz również zbiór nieobecni, zawierający elementy {'Iza’,’Piotrek’}. Następnie utwórz połączenie zbiorów i odejmij od niego zbiór nieobecni.

spis1 = {'Ala','Tomek','Iza','Piotrek'}
spis2 = {'Tomek','Beata','Kamil'}
nieobecni = {'Iza','Piotrek'}
goscie = spis1.union(spis2)
goscie = goscie.difference(nieobecni)
print(goscie)
