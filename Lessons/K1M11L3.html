Funkcje dla słowników

Poznałeś już jedną funkcję dla słownika – update(), która uaktualnia słownik o inny słownik podany jako parametr funkcji oraz operator del, który usuwa klucz ze słownika. W języku Python zdefiniowanych zostało jeszcze kilka użytecznych funkcji i operatorów. Bardzo pożytecznym operatorem jest operator in, który sprawdza czy podany klucz występuje w słowniku:

slownik = { 600123 : 'Ala', 508228 : 'Tomek', 604725 : 'Kasia' }
print(508228 in slownik) True

Wykorzystując znany z poprzednich lekcji operator warunkowy if w połączeniu operatorem in, możesz wypisać komunikat w przypadku braku podanego klucza:

klucz_do_znalezienia = 608888
slownik = { 600123 : 'Ala', 508228 : 'Tomek', 604725 : 'Kasia' }
if klucz_do_znalezienia in slownik:
print(slownik[klucz_do_znalezienia])
else:
print('Nie znaleziono klucza!'))
Nie znaleziono klucza!

Info: W starej wersji języka Python (2.x) istniała funkcja has_key(), jednak w Python 3 została ona została operatorem in.

Aby zwrócić wszystkie dostępne klucze w słowniku w postaci listy możesz użyć funkcji keys()

slownik = { 600123 : 'Ala', 508228 : 'Tomek', 604725 : 'Kasia' }
slownik.keys() [508228, 600123, 604725]

Analogicznie możesz wykorzystując funkcję values() możesz zwrócić wszystkie wartości ze słownika w postaci listy:

slownik = { 600123 : 'Ala', 508228 : 'Tomek', 604725 : 'Kasia' }
slownik.values() ['Tomek', 'Ala', 'Kasia']

Możesz wykorzystać także funkcję items(), która zwróci listę, składającą się z krotek zawierających pary klucz : wartość:

slownik = [(508228, 'Tomek'), (600123, 'Ala'), (604725, 'Kasia')]
slownik.items() [(508228, 'Tomek'), (600123, 'Ala'), (604725, 'Kasia')]

Inna funkcja o nazwie clear() pozwoli na usunięcie wszystkich elementów ze słownika:

slownik = { 600123 : 'Ala', 508228 : 'Tomek', 604725 : 'Kasia' }
slownik.clear()
print(slownik) {}

Info: Jeżeli chcesz utworzyć pusty słownik, to przypisz do niego puste nawiasy klamrowe.

A jak sprawdzić ile par klucz : wartość znajduje się w słowniku? Podobnie jak dla listy, krotki czy łańcucha znakowego możesz wykorzystać funkcję len()

slownik = { 600123 : 'Ala', 508228 : 'Tomek', 604725 : 'Kasia' }
print(len(slownik)) 3

Sprawdź czy potrafisz

Utwórz książkę telefoniczną, pozwalającą na wyszukiwanie numerów do znajomych: Adam – 800232, Kamil – 333585, Seba – 222111. Napisz program, który pobierze nazwę abonenta od użytkownika (okienko), np. Tomek i jeśli jest, wypisze numer telefonu do niego. Przetestuj działanie programu podając imię ‘Adam’.

Jak to zrobić?

Utwórz słownik o nazwie telefony z parami osoba : numer telefonu. Następnie do zmiennej np. osoba przypisz wartość funkcji input(‘Podaj abonenta’). Po czy stosując operatory if oraz in sprawdź czy istnieje klucz pobrany do zmiennej osoba i jeśli tak, to go wypisz telefony[osoba].

telefony = {'Adam':800232,'Kamil':333585,'Seba':222111}
osoba=input('Podaj abonenta')
if osoba in telefony.has_key(osoba):
print(telefony[osoba])
