Zaokrąglanie liczb

W języku Python do zaokrąglania liczb możemy wykorzystać funkcję round(), która zaokrągla liczbę dziesiętną do liczby całkowitej. Funkcja ta nie wymaga importu modułu math!

round(2.35)
2.0

Funkcja round() umożliwia także zaokrąglanie liczb do podanej liczby miejsc po przecinku. Liczbę miejsc należy przekazać jako drugi argument funkcji:

round(2.35,1)
2.4

Możesz także zamienić liczbę dziesiętną na liczbę całkowitą. Taka operacja może być wymagana jeżeli np. funkcja przyjmuję jako argumenty tylko liczby całkowite.

Do przekształcania liczby rzeczywistej (float) na liczbę całkowitą (int) służy funkcja int():

int(2.35)
2

Zwróć uwagę, że Python tym razem wypisze 2 (liczba całkowita), a nie 2.0 (liczba rzeczywista).

Info: W języku Python dla uproszczenia określono tylko dwa typy liczb: całkowity (z ang. integer), zapisywany w skrócie jako int oraz rzeczywisty, inaczej zmiennoprzecinkowy (z ang. floating point), zapisywany jako float.

Liczbę całkowitą możesz zamienić na rzeczywistą funkcją float():

float(2)
2.0

Sprawdź czy potrafisz

Zaokrąglij liczbę PI z biblioteki math do 3 miejsc po przecinku.
Jak to zrobić?

Zaimportuj moduł math, a wykorzystaj funkcję round z parametrami math.pi oraz 3.

import math
round(math.pi,3)
