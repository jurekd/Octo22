Funkcje w klasach

W poprzedniej lekcji zawartość obiektów wypisywaliśmy podając każdą ich daną z osobna:

osoba1 = Osoba("Kamil","Bilski",15)
print(osoba1.imie,osoba1.nazwisko,osoba1.klasa)
Kamil Bilski 15

Jest to stosunkowo niewygodne, bo co jeśli tych danych będzie naprawdę dużo? Na szczęście oprócz danych i konstruktorów w klasie można definiować także funkcje, które operują na danych. Utwórzmy zatem funkcję wypisz(), która wypisze wszystkie potrzebne dane:

class Osoba:
imie = "imie"
nazwisko = "nazwisko"
wiek = 0
def __init__(self, imie, nazwisko, wiek):
self.imie = imie
self.nazwisko = nazwisko
self.wiek = wiek
def wypisz(self):
print(self.imie,self.nazwisko,self.wiek)
osoba1 = Osoba("Kamil","Bilski",15)
osoba1.wypisz()
Kamil Bilski 15

Jeszcze wygodniej byłoby napisać print(osoba1). Jeżeli tego nie zmienisz, zostanie wypisana referencja (adres w pamięci) obiektu:

print(osoba1)

Aby zmienić to domyśle zachowanie, możesz w klasie zdefiniować funkcję __str__(self), której zadaniem jest automatyczna zamiana obiektu na tekst:

class Osoba:
# tutaj kod klasy jak poprzednio
def __str__(self):
return self.imie+' '+self.nazwisko+' '+str(self.wiek)
osoba1 = Osoba("Kamil","Bilski",15)
print(osoba1)
Kamil Bilski 15

Funkcje klasy, oprócz wypisywania zawartych w nich danych, mogą realizować różne zadania. Dopiszmy funkcję inicjały, która wyświetla pierwszą literę imienia i nazwiska:

class Osoba:
# tutaj kod klasy jak poprzednio
def inicjaly(self):
return self.imie[0]+self.nazwisko[0]
print(osoba1.inicjaly())
KB

Tym razem stworzyliśmy funkcję, która zwraca ciąg znaków, zatem żeby go wyświetlić, musimy użyć funkcji print().

Na koniec tej lekcji jeszcze jedna funkcja skonczone(), która sprawdza czy dana osoba skończyła podaną liczbę lat.

class Osoba:
# tutaj kod klasy jak poprzednio
def skonczone(self, ile):
if self.wiek>=ile:
return True
else:
return False
osoba1 = Osoba("Kamil","Bilski",15)
if osoba1.skonczone(16):
print("Masz dostęp")
else:
print("Nie masz dostępu")
Nie masz dostępu

Sprawdź czy potrafisz

Do klasy komórka z poprzedniej lekcji, dopisz funkcję __str__(self), która opisze komórkę w postaci np. Samsung S7 2016 600200300 oraz funkcję wiek, która na podstawie roku produkcji i bieżącego roku liczy ile lat ma dana komórka.

Jak to zrobić?

W class Komorka dopisz funkcję __str__(self), która zwraca ciąg składający się z producenta, modelu, roku produkcji i numeru. Pamiętaj o zamianie tych dwóch ostatnich danych na tekst funkcją str(). Następnie dopisz funkcję wiek(self), w której odczytaj bieżący rok z modułu datetime: rok = datetime.date.today().year. Odejmij od niego rok produkcji i zwróć w return. Wypisz dane komórki oraz jej wiek funkcją print.

import datetime
class Komorka:
producent=""
model=""
rok_produkcji=0
def __init__( self, producent, model, rok_produkcji):
self.producent = producent
self.model = model
self.rok_produkcji = rok_produkcji
def __str__(self):
return self.producent+' '+self.model+' '+str(self.rok_produkcji);
def wiek(self):
rok = datetime.date.today().year
return rok-self.rok_produkcji
tel1 = Komorka("Samsung", "S7", 2016)
print(tel1,tel1.wiek())
