Rozszerzanie i zmiana list

Przechowywanie danych w liście przynosi sporo korzyści. Wiesz już, że dzięki temu możesz przechowywać dane pod wspólną zmienną, w łatwy powiększać je oraz usuwać z nich elementy. Python dostarcza też spory zestaw użytecznych funkcji, dzięki którym wykonanie wielu zadań na zbiorze danych będzie bardzo proste.

Pamiętasz funkcję len() dla tekstów? Funkcja ta liczyła liczbę znaków w tekście. Analogiczna funkcja dla listy zwraca liczbę elementów w niej zawartych.

liczby = [11,8,3,17,8]
dlugosc = len(liczby)
print(dlugosc)
5

A co jeżeli chcesz znaleźć ile elementów o zadanej wartości jest na liście? Możesz wykorzystać funkcję count(), analogicznie jak dla tekstów.

liczby = [11,8,3,17,8]
ile8 = liczby.count(8)
print(ile8)
2

Jeżeli elementu nie ma na liście, zwracana jest wartość 0.

Info: Pamiętaj, że w języku Python podstawowe funkcje takie jak np. len() wywoływane są przez przekazanie wartości (lub zmiennej reprezentującej wartość) do funkcji. Większość funkcji określona jest jednak dla obiektu danych (w naszym przypadku listy) i takie funkcje wywoływane są bezpośrednio na zmiennej lub wartości. Funkcje tego typu wywołujemy umieszczając ich nazwę po kropce, doklejonej zaraz po nazwie zmiennej.

Inną pożyteczną funkcją dla list jest funkcja min(), zwracająca najmniejszy element listy:

liczby = [11,8,3,17,8]
najmniejsza = min(lista1)
print(najmniejsza)
3

Analogicznie funkcja max() zwraca największy element z listy:

liczby = [11,8,3,17,8]
najwieksza = max(liczby)
print(najwieksza)
11

Sprawdź czy potrafisz

Przesuń największy element z listy [1,4,5,3,2] na jej koniec.

Jak to zrobić?

Utwórz listę liczby i wpisz do niej elementy 1, 4, 5, 3 i 2. Następnie znajdź element największy, zapamiętaj go pod zmienną największy. W kolejnym kroku usuń go z listy, a na koniec zmienioną listę powiększ o zapamiętany element i wypisz ją.

liczby=[1,4,5,3,2]
najwiekszy=max(liczby)
liczby.remove(najwiekszy)
liczby.append(najwiekszy)
print(liczby)
