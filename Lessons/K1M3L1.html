Zapis i łączenie tekstów

W języku programowania Python możesz przetwarzać dane nie tylko liczbowe. W prosty sposób możesz działać również na tekstach.

Info: Żeby odróżnić teksty od liczb wykorzystywany jest cudzysłów (") lub apostrof (').

Wprowadźmy teksty:

"Python jest prosty"
'Python jest prosty' 'Python jest prosty'
'Python jest prosty'

Oba sposoby są dobre, ale Python wyniki tekstowe będzie wypisywał zawsze w apostrofach.

Na tekstach możesz wykonywać operację dodawania – wynikiem będzie złączenie tekstów:

'Python jest prosty ' + 'i szybko go poznam.'
'Python jest prosty i szybko go poznam.'

Nie możesz dodawać tekstów do liczb. Poniższe działanie spowoduje błąd:

'Ala'+5
TypeError: cannot concatenate 'str' and 'int' objects

Info: Powyższy błąd oznacza, że Python nie może złączyć tekstu i liczby (Dosłownie: "BłądTypu: nie mogę złączyć obiektów 'łańcuch znakowy' i 'liczba całkowita'").

Zamiany liczby na tekst możesz dokonać wykorzystując funkcję o nazwie str(). Ta funkcja nie wymaga importowania żadnego modułu!

'Ala'+ str(5)
'Ala5'

Sprawdź czy potrafisz

Wypisz wynik działania 2+3 w postaci: 2 dodać 3 to 5.
Jak to zrobić?

Wypisz tekst "2 dodać 3 to " i dodaj do niego funkcję str(2+3)

'2 dodać 3 to '+str(2+3)
