Definiowanie wyrażeń lambda

Wyrażenie lambda definiujemy słowem lambda, po którym umieszczamy parametry, a po dwukropku operacje, które mają być na tych parametrach wykonane:

lambda x: x * x (3)
9

Takie proste funkcje możemy przypisywać do zmiennych i późnej wywoływać je podając nazwę zmiennej:

suma = lambda a, b: a + b
print(suma(2,3)
5

Powyższy zapis jest niemal jednoznaczny z funkcją zapinaną jako:

def suma(a, b):
return a+b

print(suma(2,3)
5

Lambada wyrażenia są krótsze w zapisie, ale mają sporo ograniczeń: nie mogą definiować własnych zmiennych, nie mogą zwierać instrukcji return, nie mogą też zawierać więcej niż jednej instrukcji.

Jednak dzięki tym ograniczeniom mogą stać się częścią standardowych funkcji definiowanych słowem def:

def suma_kwadratow(lista):
kwadrat = lambda x: x * x
suma = 0
for element in lista:
suma = suma + kwadrat(element)
return suma

wynik = suma_kwadratow([1,2,3])
print(wynik)
14

Lambda wyrażenia można też przekazywać jako parametry funkcji

def oblicz(funkcja, a, b):
wynik = funkcja(a, b)
print(wynik)

dodaj = lambda a, b: a+b
oblicz(dodaj, 2, 3)
5
pomnoz = lambda a, b: a/b
oblicz(pomnoz, 2, 3)
6

Funkcja może też zwracać lambda wyrażenie:

def funkcja(x):
return lambda y : x * y

print(funkcja(2)(3))
6

W powyższym przykładzie funkcja wywoływana jest z argumentem 2 i zwracane jest wyrażenie lambda y : 2 * y, do którego z kolei przekazana zostaje wartość 3, zatem ostateczny wynik wynosi 6. Staraj się jednak unikać tak złożonych konstrukcji, o ile to nie jest z jakiegoś powodu uzasadnione.

Sprawdź czy potrafisz

Stosując wyrażenie lambda napisz funkcję suma_kwadratow, która oblicza sumę kwadratów trzech podanych liczb. Przetestuj funkcję dla wartości 1,2,3.

Jak to zrobić?

Do zmiennej suma_kwadratow przypisz lambda wyrażenie, które pobiera parametry a, b, c, a po dwukropku wpisz wyrażenie a*a + b*b + c*c. Następnie wywołaj suma_kwadratow podając jako argumenty liczby 1, 2, 3

suma_kwadratow = lambda a, b, c: a*a + b*b + c*c
wynik = suma_kwadratow(1, 2, 3)
print(wynik)
