Używanie lambda wyrażeń w listach

Lambda wyrażenia są bardzo pomocne w szybkim filtrowaniu list. Przykładowo jeżeli z podanej listy chcemy zostawić tylko te elementy, które są dodatnie możemy użyć pętli

lista = [ 1, 2 ,-1, 3, -3]
nowa = []
for element in lista:
if element>0:
nowa.append(element)
print(nowa)
[1, 2, 3]

Ale możemy także wykorzystać funkcję filtr() z lambda wyrażeniem, co jest znacznie krótsze.

lista = [ 1, 2 ,-1, 3, -3]
nowa = filter(lambda x: x>0, lista)
print(nowa)
[1, 2, 3]

Filtry możemy też wykorzystać do zwrotu części wspólnej dwóch list (analogicznie jak funkcja intersection):

lista1 = [1,2,3,5]
lista2 = [2,3,4,6]
print(filter(lambda x: x in lista1, lista2))
[2, 3]

Oprócz filtrowania, wyrażenia lambda możesz wykorzystać do zamiany elementów listy na elementy, które zostaną przekształcone, zgodnie z wyrażeniem lambda. Przykładowo dla listy liczb utworzymy listę kwadratów tych liczb:

liczby = range(1,6) # tworzymy [1, 2, 3, 4, 5]
kwadraty = map(lambda x: x*x, liczby)
print(kwadraty)
[1, 4, 9, 16, 25]

Powyższy kod za pomocą zwykłej pętli zapisalibyśmy tak:

liczby = range(1, 6)
kwadraty = []
for x in liczby:
kwadraty.append(x*x)
print(kwadraty)
[1, 4, 9, 16, 25]

Inny przykład – dla listy liczb utworzymy mapę zawierającą True, jeśli liczba jest parzysta i False, jeśli liczba nie jest parzysta.

liczby = [1, 3, 4, 5, 8]
parzyste = map(lambda x: x%2==0, liczby)
print(parzyste)
[False, False, True, False, True]

Ostatnią ważną funkcją, która umożliwia szybkie przetwarzanie list jest funkcja reduce(). Jej zadaniem jest wykonanie podanej funkcji na wszystkich elementach listy i zwróceniu jej wyniku:

lista = [1,3,5,7]
redukcja=reduce(lambda x, y: x + y, lista)
print(redukcja)
16

Wyrażenie lambda dla reduce() musi przyjmować dwa parametry. Redukcja działa w ten sposób, że najpierw brane są 2 pierwsze elementy i dla nich obliczany jest wynik. Następnie brany jest wynik i kolejny element, i tak dalej.

Sprawdź czy potrafisz

Z listy telefony=['200123', '404111', '222333a', 'b500707', '120120'] wybierz tylko te, które zawierają wyłącznie cyfry. Do sprawdzenia możesz wykorzystać funkcję isdigit().

Jak to zrobić?

Zdefiniuj listę. Następnie określ filtr, który przekazuje lambda wyrażenie x : x.isdigit() oraz listę telefonów. Wypisz listę utworzoną w wyniku działania filtru.

lista = ['200123', '404111', '222333a', 'b500707', '120120']
telefony = filter(lambda x: x.isdigit(), lista)
print(telefony)
